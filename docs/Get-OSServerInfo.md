# Get-OSServerInfo

## SYNOPSIS
Returns a summary information about the OutSystems Server

## SYNTAX

```
Get-OSServerInfo [<CommonParameters>]
```

## DESCRIPTION
This will returns the OutSystems Server version, install directory, serial number, machine name and private key.

## EXAMPLES

### EXAMPLE 1
```
Get-OSServerInfo
```
