# Disable-OSIPv6

## SYNOPSIS
Disable IPv6.

## SYNTAX

```
Disable-OSIPv6 [<CommonParameters>]
```

## DESCRIPTION
This will disable IPv6 on the server.
IPv6 is not supported by Outsystems
