# Start-OSServerServices

## SYNOPSIS
Starts the OutSystems platform services.

## SYNTAX

```
Start-OSServerServices [<CommonParameters>]
```

## DESCRIPTION
This will start all OutSystems platform services by the recommended order.

## EXAMPLES

### EXAMPLE 1
```
Start-OSServerServices
```
