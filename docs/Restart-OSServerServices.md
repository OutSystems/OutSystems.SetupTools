# Restart-OSServerServices

## SYNOPSIS
Restarts OutSystems services.

## SYNTAX

```
Restart-OSServerServices [<CommonParameters>]
```

## DESCRIPTION
This will restart all OutSystems platform services by the recommended order.

## EXAMPLES

### EXAMPLE 1
```
Restart-OSServerServices
```
