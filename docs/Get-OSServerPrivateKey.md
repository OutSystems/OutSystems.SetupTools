# Get-OSServerPrivateKey

## SYNOPSIS
Returns the OutSystems platform server private key.

## SYNTAX

```
Get-OSServerPrivateKey [<CommonParameters>]
```

## DESCRIPTION
This will returns the OutSystems platform server private key.

## EXAMPLES

### EXAMPLE 1
```
Get-OSServerPrivateKey
```
