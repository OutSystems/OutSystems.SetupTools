# Get-OSServerInstallDir

## SYNOPSIS
Returns where the OutSystems platform server is installed.

## SYNTAX

```
Get-OSServerInstallDir [<CommonParameters>]
```

## DESCRIPTION
This will returns where the OutSystems platform server is installed.

## EXAMPLES

### EXAMPLE 1
```
Get-OSServerInstallDir
```
