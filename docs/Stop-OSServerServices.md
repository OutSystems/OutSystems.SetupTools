# Stop-OSServerServices

## SYNOPSIS
Stops the OutSystems platform services.

## SYNTAX

```
Stop-OSServerServices [<CommonParameters>]
```

## DESCRIPTION
This will stop all Outsystems platform services by the recommended order.

## EXAMPLES

### EXAMPLE 1
```
Start-OSServerServices
```
