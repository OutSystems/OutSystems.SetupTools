# Get-OSServerVersion

## SYNOPSIS
Returns the OutSystems platform server version

## SYNTAX

```
Get-OSServerVersion [<CommonParameters>]
```

## DESCRIPTION
This will returns the OutSystems platform server version.

## EXAMPLES
